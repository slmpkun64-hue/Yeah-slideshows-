Connect your Google Drive

1

Go to Google Cloud Console, create a project, enable the Google Drive API and Google Picker API.

2

Create an OAuth 2.0 Client ID (Web application). Add this page's URL as an authorised JavaScript origin. Also create an API Key and restrict it to Drive & Picker APIs.

3

Paste your credentials below and click Connect.

Please enter both API Key and Client ID.
